This project visualizes data from the US 2020 Election, utilizing datasets from Kaggle and voter turnout information from [Elect Project](http://www.electproject.org/home/voter-turnout/voter-turnout-data). Additional resources include US State Population data and a public release crosstabs Excel file. The project employs Vegalite and Echarts for interactive and insightful visualizations.![](/vegalite-us.png)
